📊 BANCO DE DADOS (4 tabelas)
  
  User
  ├─ id (UUID)
  ├─ cpf (unique)
  ├─ name
  ├─ email (unique)
  └─ phone (opcional)

  Raffle
  ├─ id (UUID)
  ├─ title
  ├─ image (url)
  ├─ images[] (até 20)
  ├─ prizeAmount
  ├─ totalQuotas
  ├─ soldQuotas
  ├─ quotaPrice (0.50)
  ├─ status (open/closed/drawn)
  ├─ winner (CPF do vencedor)
  └─ creatorId (FK)

  RafflePurchase
  ├─ id
  ├─ userId (FK)
  ├─ raffleId (FK)
  ├─ quotas (quantidade)
  ├─ amount (valor pago)
  ├─ numbers (JSON array)
  └─ status (pending/completed)

  TopBuyer
  ├─ userId (unique)
  ├─ totalSpent
  ├─ totalQuotas
  └─ raffleBought

🚀 COMO COMEÇAR

  1. npm install
  2. npx prisma dev (copiar URL)
  3. Editar .env com DATABASE_URL
  4. npx prisma migrate dev --name init
  5. npm run prisma:seed (opcional - dados teste)
  6. npm run dev
  7. Abrir http://localhost:3000

⚡ COMANDOS ÚTEIS

  npm run dev              - Iniciar dev server
  npx prisma studio       - Ver BD visualmente (5555)
  npx prisma migrate dev   - Criar migração
  npm run prisma:seed     - Carregar dados de teste
  npm run build           - Build para produção
  npm run start           - Iniciar servidor prod
